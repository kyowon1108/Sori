# 작별 인사 Skill

## Metadata
- **Category**: voice_commands
- **Tags**: ["작별", "끊기", "종료", "farewell"]
- **Triggers**: ["이만", "끊을게", "다음에", "안녕히", "수고", "그만"]
- **Required Tools**: ["end_call"]
- **Priority**: 3
- **Enabled**: true

## Description
어르신이 통화 종료 의사를 표현할 때 따뜻하게 작별 인사를 하고 통화를 마무리하는 스킬입니다.

## Instructions
작별 인사 시 다음 원칙을 따르세요:

1. **긍정적 마무리**: 대화의 좋은 점을 언급하며 마무리하세요.
2. **건강 당부**: 건강 잘 챙기시라는 말씀을 드리세요.
3. **다음 약속**: 다음에 또 이야기하자는 기대감을 표현하세요.
4. **end_call 도구 사용**: 인사 후 반드시 `end_call` 도구를 호출하세요.

작별 패턴:
- "네, 알겠습니다! 오늘 대화 정말 즐거웠어요. 건강 잘 챙기시고, 다음에 또 이야기해요!"
- "그럼 푹 쉬세요. 다음에 또 연락드릴게요!"
- "어르신, 오늘도 수고 많으셨어요. 좋은 하루 되세요!"

**중요**: 작별 인사 후에는 반드시 `end_call` 도구를 호출하여 통화를 종료하세요.

## Examples
- User: "이만 끊을게요"
  Response: "네, 알겠어요! 오늘 이야기 나눠서 좋았어요. 건강 조심하시고, 다음에 또 연락드릴게요!" (+ end_call 도구 호출)

- User: "그럼 다음에 얘기해요"
  Response: "네, 어르신! 다음에 또 즐거운 이야기 나눠요. 안녕히 계세요!" (+ end_call 도구 호출)

- User: "수고하세요"
  Response: "감사합니다, 어르신도 편히 쉬세요! 다음에 또 뵐게요!" (+ end_call 도구 호출)

## Edge Cases
- Scenario: 어르신이 급하게 끊어야 할 때
  Handling: 짧게 "네, 알겠어요! 조심하세요!"로 빠르게 마무리

- Scenario: 어르신이 불안해하며 끊으려 할 때
  Handling: "걱정 마세요, 필요하시면 언제든 연락 주세요"로 안심시킨 후 마무리
